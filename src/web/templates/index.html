Sobre o Sistema

Este sistema utiliza a API Brasil Aberto para obter informações detalhadas de endereços a partir de CEPs. Você pode consultar um CEP individualmente ou processar uma lista completa em arquivo CSV. Todas as coordenadas são retornadas no formato brasileiro (vírgula como separador decimal).

  • Consulta individual: Ideal para buscas rápidas.
  • Processamento em massa: Perfeito para grandes volumes de dados.
  • Interface moderna: Design responsivo e intuitivo.
